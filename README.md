# Beauty Nails & Hair - Online Booking Website

A fully responsive online booking platform for a nail and hair salon, allowing customers to select services, choose dates and times, and confirm payments via WhatsApp. Admins can view all bookings in a dedicated dashboard.

## Live Demo

[View the Website](https://huthkid01.github.io/beauty-nails-and-hair/)

## Features

- **Service Selection:** Users can choose one or multiple services.
- **Dynamic Pricing:** Total cost updates automatically based on selected services.
- **Booking Form:** Collects name, email, phone, date, and time.
- **Payment Confirmation:** Redirects users to WhatsApp for payment confirmation.
- **Admin Dashboard:** View all bookings with details and total count.
- **Responsive Design:** Optimized for mobile, tablet, and desktop.
- **Dark Mode Support:** Easily adaptable via CSS variables.

## Technologies Used

- **HTML5** – Markup for structure  
- **CSS3** – Styling, animations, and responsive design  
- **JavaScript (ES6)** – Interactivity, DOM manipulation, and Supabase integration  
- **Supabase** – Backend for storing bookings  
- **GitHub Pages** – Hosting the website live
